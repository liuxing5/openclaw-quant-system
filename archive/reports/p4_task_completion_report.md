#### **回放引擎功能**
1. **历史策略回放**: 在任意历史时间点重新执行策略
2. **策略快照**: 保存策略状态，支持随时回放
3. **参数敏感性分析**: 不同参数配置下的表现对比
4. **多策略对比**: 多个策略版本的横向比较
5. **综合评分**: 0-100分策略质量评分系统
6. **专业报告**: 生成详细的回放分析报告

#### **回放流程**
```
策略快照 → 历史数据加载 → 参数配置 → 回放执行 → 
结果分析 → 对比评估 → 报告生成
```

#### **应用场景**
- **策略优化**: 测试不同参数配置的效果
- **历史验证**: 验证策略在历史极端情况下的表现
- **版本对比**: 比较策略不同版本的改进效果
- **参数调优**: 找到最优参数组合
- **风险分析**: 评估策略的风险特征

## 🚀 立即使用指南

### **1. 快速启动 (开发模式)**
```bash
# 克隆代码
git clone https://github.com/liuxing5/openclaw-quant-system.git
cd openclaw-quant-system

# 安装依赖
pip install -r requirements.txt

# 启动集成应用
cd quant_system/web_dashboard
python app_integrated.py
```

### **2. Docker部署 (生产模式)**
```bash
# 单容器部署
docker build -t quant-system .
docker run -p 5000:5000 quant-system

# 多服务部署
docker-compose up -d
```

### **3. 初始化系统**
```bash
# 访问API初始化权限系统
curl -X POST http://localhost:5000/api/auth/initialize

# 默认管理员账户
用户名: admin
密码: admin123 (首次登录后必须修改)
```

### **4. API使用示例**
```python
import requests

# 用户登录
response = requests.post('http://localhost:5000/api/auth/login', json={
    'username': 'admin',
    'password': 'admin123'
})
token = response.json()['access_token']

# 调用受保护的API
headers = {'Authorization': f'Bearer {token}'}
response = requests.get('http://localhost:5000/api/dashboard/stats', headers=headers)
```
